About WireGuard Interfaces:
A WireGuard interface is a virtual network interface that will handle VPN traffic. You can have multiple interfaces, each with its own configuration and peers.

Must contain only letters, numbers, and underscores (no spaces or special characters)
Examples: wg0, wg1, vpn_production, vpn_test

The UDP port that WireGuard will listen on
Default: 51820 (change if port is already in use)

Port Note:
Make sure this port is:
  • Not already in use by another service
  • Open in your firewall (if applicable)
  • Forwarded on your router (if hosting at home)
