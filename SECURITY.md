# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| > 0.1.1   | :white_check_mark: |
| < 0.1.0   | :x:                |

## Reporting a Vulnerability

Please report any vulnerabilities via [GitHubs private security vulnerability reporting](https://docs.github.com/en/code-security/how-tos/report-and-fix-vulnerabilities/privately-reporting-a-security-vulnerability) or mail at: wg-manager@untersander.dev
